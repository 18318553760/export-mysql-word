# export-mysql-word

利用python制作msql数据库字典

使用教程：

1、安装python,默认安装了pip

2、进入cmd 运行pip install PyMysql

3、pip install python-docx

4、python mysql.py

上面的安装是否成功，cmd--python--import PyMysql

为了确保成功，请先在目录下建立文件

Python读写docx文件教程:	http://yshblog.com/blog/40

官方教程https://python-docx.readthedocs.io/en/latest/
